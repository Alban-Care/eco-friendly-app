Objectif de l’application

Créer une plateforme collaborative en ligne pour promouvoir l'écologie et l'écocitoyenneté.

Rôles utilisateus

  • Visiteurs
  • Membres (utilisateurs inscrits)
  • Auteurs
  • Administrateur

Après les spé fonctionnelles :

  • mise en place des user-stories sur Trello
  • création des wireframes avec whimsical

Spécifications fonctionnelles

front-end :

  1. Création d'une page d'accueil
  2. Création d'une page pour chaque catégorie
  3. Création d'une page pour tous les articles
  4. Création d'une page pour tous les conseils
  5. Création d'une page pour chaque article et conseil
  6. Création d'une page d'inscription et de connexion
  7. Création d'une page de profil
  8. Création d'un formulaire d'ajout/édition de conseil
  9. Possibilité de rechercher et/ou fitrer du contenu
  10. Possibilité de gérer les conseils

Spécifications fonctionnelles

back-end :

  1. Création d'une page de connexion
  2. Création d'une page d’accueil administrateur
  3. Création d'une page gestion de contenu administrateur
  4. Création d'une page gestion de contenu auteur
  5. Possibilité de stocker des informations via une API
  6. Possibilité de récupérer des informations via une API
